Pandora
=======

Bukkit plugin with many miscellaneous features specifically created for Addstar MC
Current Integrations:
  * Monolith
  * PhatLoots
  * Worldedit
  * WorldGuard
  * QuickShop
  * Votifier
  * LWC
  * VanishNoPacket
  * Blockhunt
  * Minigames
  * SurvivalGames
  * Citizens
  * Treasures
  * LibsDisguises
